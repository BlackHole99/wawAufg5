Aufgabe 2

Das Problem ist, dass auf einem Computer mehrere Anwendungen gleichzeitig Zugriff auf das World Wide Web haben wollen.
Hierfür wird der Port verwendet. Jede Anwendung hat hierfür einen eigenen (teils zufälligen) Port über den kommuniziert wird.
Der Browser benötigt für die Antwort des Webservers ebenfalls einen Port. Dieser wird vom Betriebssystem zugeteilt und ist für
jedes Browserfenster (-tab) unterschiedlich.
